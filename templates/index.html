Home Pricing Prediction

Project Overview

The goal of this project was an attempt to understand Portland, OR home pricing through machine learning. More specifically, we developed a statistical model and analysis that helped us understand the relationship between house features and home price. We created multiple models using various data sets and features to see how these effected eachother.

Our analysis was predicated on the potential to predict house prices with these variables through a user application. This application would not predict a selling price, but more important it would predict the house value. We found this to be directionally accurate and also came to many conclusions and learnings through the process.

Data Exploration

Linear Regression

The initial model we created was based on multiple linear regression using the basic features of a home (No. of Beds/Baths, Square feet, Lot Size, Latitude/Longitude).

After creating this model we acheived very low scores. This initial analysis was built on very limited data and only using the four main variables stated above. We then expanded our data set and variables, almost doubling both and drastically improving our score by over 15%.

We concluded that we still needed more data and more variables. This was a main constraint of this project as Real Estate data is protected and difficult to obtain. We explored many more creative ways we could use the data we already had (Centralization points, Street Name vs Number, School Zones, West vs East).

Deep Learning Model

Another possibility we considered was running a deep learning model on another vetted dataset. This dataset was used as a kaggle competition and was actual data with additional variables we did not have access too. The main reason for doing this was to compare our simple data/variable linear regression model with a deep learning model which used over double the data points and variables. This would be a good indicator of how successful our model was or could be.

To our surprise, the deep learning model was only 9% better. This solidified our approach to creating a model based on linear regression and our limited dataset but also trying to improve our varaiables.

House Price Prediction

Our final product will use our linear regression model and help users predict the value of their home.
